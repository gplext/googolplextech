# Googolplex Tech Website

A modern, responsive static website for **Googolplex Tech | Innovation Beyond Infinity**.

## Running Locally with Docker

You can easily run and preview this website locally using Docker Compose. The environment is configured to run on port **3081**.

### Prerequisites

Make sure you have **Docker** and **Docker Compose** installed on your system.

### Instructions

1. **Start the server**:
   Run the following command in the root directory:
   ```bash
   docker compose up -d
   ```

2. **Access the website**:
   Open your browser and navigate to:
   [http://localhost:3081](http://localhost:3081)

3. **Stop the server**:
   To shut down the container, run:
   ```bash
   docker compose down
   ```
